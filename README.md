# C ⚙️

My collection of C files.

## How to compile.

**Please compile against [GCC](https://gcc.gnu.org/)! [Clang](https://clang.llvm.org/) is not currently tested, sorry.**

`gcc filename.c -o executable.out`

## How to run: 

Mark as executable.

`chmod +x executable.out`

Run.

`./executable.out`
